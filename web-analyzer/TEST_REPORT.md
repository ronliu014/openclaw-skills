## 🔧 技术实现

### 核心工具
- `browser(action="open")` - 打开页面
- `browser(action="snapshot")` - 捕获内容
- `browser(action="screenshot")` - 截图
- `browser(action="start")` - 启动浏览器

### 数据流
1. 用户请求 → 触发 skill
2. 启动/检查浏览器状态
3. 打开目标 URL
4. 等待页面加载（3-5秒）
5. 捕获 AI 格式快照
6. 解析和提取内容
7. 生成结构化输出

### 错误处理
- 浏览器未启动 → 自动启动
- 页面加载超时 → 增加等待时间
- 内容提取失败 → 回退到 web_fetch
- 连接失败 → 提示重启 gateway

## 📊 性能指标

- **启动时间**: ~2-3秒（浏览器已运行）
- **页面加载**: 3-5秒（复杂页面）
- **内容提取**: <1秒
- **总耗时**: 约 5-10秒/页面

### OpenClaw 浏览器工具
- **profile="openclaw"** - 隔离环境，不影响用户浏览器
- **profile="chrome"** - 使用用户的 Chrome（需要扩展）
- **snapshotFormat="ai"** - AI 优化的文本格式
- **targetId** - 标签页 ID，用于后续操作
